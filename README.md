# redrock-homework
红岩网校寒假Android作业

项目介绍
这是一个基于Java开发的简易资讯浏览APP，使用玩Android开放API，实现文章列表展示、网络请求、显示文章标题、作者和时间。

- 开发语言：Java
- 最低兼容：Android 8.0（API 26）
- 网络框架：Retrofit
- 列表控件：RecyclerView

功能
- 展示文章列表
- 网络请求获取数据
- 显示标题、作者、发布时间

开发心得
通过这次作业，我学会了安卓基本页面搭建、网络请求的使用，对RecyclerView和Retrofit有了初步了解，也熟悉了项目结构和GitHub提交的整个流程。
